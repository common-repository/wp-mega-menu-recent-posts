Show automatically recent posts in column in dropdown of megamenu.
 
== Description ==

 WP Mega Menu Recent Posts plugin show recent posts under dropdown of menu in grid system. You can show text rollover effect after hover on image. This is the good solution for showing recent menu in grid using Mega Menu.
 
== Features ==
 
*   You could recent posts of any category under dropdown menu.
*   Show post title and read more link on hover of image.
*   Easily set to show no of posts in row and total posts.
*   Show posts in grid system.
*   Easily configure with Mega Menu plugin
 

== Installation ==
   
1. Log-in to your WordPress admin panel. 
2. Go to plugin and click on Add New button.
3. Upload plugin's Zip file from your PC and activate plugin.
4. Now You could see "Mega Menu Recent Posts" widget under "Widgets" Section in "Appearance" menu.
